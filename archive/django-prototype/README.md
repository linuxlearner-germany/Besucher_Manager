# Django-Prototyp

Dieser Ordner enthaelt den bisherigen Django-/Python-Stand des Projekts.

Er bleibt ausschliesslich als fachliche Referenz erhalten:

- Masken
- Felder
- Prozessablauf
- Drucklayout-Ideen

Er ist nicht mehr Teil der aktiven Zielarchitektur. Die aktive Anwendung liegt unter `apps/backend` und `apps/frontend`.
